## 3. 확인 필요 사항 ⚠️

전달받은 URL을 검토한 결과 아래 항목들이 의심스러워 원본 QR 재확인이 필요합니다.

### 3.1 다수 URL이 동일한 영상 ID(`LAh9jpgcHM8`)를 가리킴
- 1_1.2 "숨 고르기 명상"의 영상 ID와 1_2.1 ~ 1_6.2, 7Core 페이지가 모두 같은 `v=LAh9jpgcHM8`로 되어 있음
- **추정**: 원본 QR을 일일이 스캔하기 어려워 임시로 같은 값을 채워주신 것으로 보임
- **요청**: 실제 페이지별 영상 ID를 다시 확인 부탁드립니다 (또는 재생목록의 index 번호 매핑이라도)

### 3.2 `index` 파라미터 불일치
- 1_2.1 ~ 1_3.1: `index=7`
- 1_3.2 ~ 7Core: `index=6`
- 1_7.1 페이지2: `index=2`
- 1_7.2: `index=3`
- → index 값이 페이지마다 들쭉날쭉해서 일관된 매핑인지 불명확

### 3.3 1_5.2 "하늘 높이 그네뛰기" URL 누락
- 색칠 전용 페이지라 영상이 원래 없을 수도 있고, QR이 존재한다면 URL이 빠진 것일 수도 있음
- 원본 지류의 1_5.2 페이지에 QR이 인쇄되어 있는지 확인 부탁드립니다

### 3.4 PDF상 페이지 번호 표기 이슈
- PDF의 오감 테라피 두 번째 페이지가 `1_2.2`인데 제목은 "방귀쟁이 며느리 시집가는 날 **1**"로 표기됨 (실제로는 "2"의 오타로 보임)
- 일기쓰기도 PDF 목차상 `1_7.1`, `1_7.2`인데 "오늘 하루 어땠나요?"가 2페이지에 걸쳐 있어 실제로는 페이지 3개로 보임
