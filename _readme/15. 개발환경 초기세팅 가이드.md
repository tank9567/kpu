## 1. 사전 준비

### 1.1 Node.js 확인

PowerShell에서 다음 명령으로 버전 확인:

```powershell
node --version
npm --version
```

- 설치되어 있지 않거나 18.x 이하라면: https://nodejs.org/ 에서 LTS 버전 설치
- 설치 후 PowerShell 재시작 필요

### 1.2 Git 확인

```powershell
git --version
```

설치되어 있지 않다면 https://git-scm.com/ 에서 설치.

### 1.3 VSCode 확인

```powershell
code --version
```

`code` 명령이 안 되면 VSCode를 열고 `Ctrl+Shift+P` → "Shell Command: Install 'code' command in PATH" 실행.

### 3.2 Vite 생성 명령

PowerShell에서 프로젝트 루트(`d:\_SRC\haru_inji`)에 위치한 상태로:

```powershell
npm create vite@latest app -- --template react-ts
```

- `app` 폴더가 새로 생성되므로 "directory is not empty" 질문 없음

### 3.3 기본 의존성 설치

```powershell

```

> ⚠️ 이후 모든 `npm`·`npx` 명령은 **`d:\_SRC\haru_inji\app\` 안에서 실행**합니다. 새 PowerShell 창을 열 때마다 `cd app` 잊지 마세요.

```powershell
npm run dev
```

브라우저에서 `http://localhost:5173/` 접속해 Vite 기본 화면이 보이면 성공. 확인 후 `Ctrl+C`로 서버 종료.

### 5.2 `tailwind.config.js` 편집

생성된 파일을 다음 내용으로 교체합니다 (어르신 친화 디자인 토큰 포함):

```js
/** @type {import('tailwindcss').Config} */
export default {
  content: ['./index.html', './src/**/*.{ts,tsx}'],
  theme: {
    extend: {
      fontSize: {
        // 어르신 친화 기본값
        base: ['1.25rem', { lineHeight: '1.7' }],   // 20px
        lg: ['1.5rem', { lineHeight: '1.6' }],      // 24px
        xl: ['1.75rem', { lineHeight: '1.5' }],     // 28px
        '2xl': ['2rem', { lineHeight: '1.4' }],     // 32px
      },
      minHeight: {
        'btn': '56px',
        'touch': '48px',
      },
      colors: {
        // PoC 단계 임시 팔레트 — 정식 디자인 단계에서 교체
        primary: {
          DEFAULT: '#D94B6B',
          dark: '#A03350',
        },
        ink: {
          DEFAULT: '#2B2B2B',
          soft: '#4A4A4A',
        },
      },
    },
  },
  plugins: [],
};
```

### 7.2 타입 정의

`src/lib/types.ts`:

```ts
export type Issue = {
  id: string;
  issueNo: number;
  weekNo: number;
  theme: string;
  publishedAt: string;
  days: Day[];
};

export type Day = {
  day: number;
  pages: Page[];
};

export type Page = {
  pageNo: string;
  category: string;
  title: string;
  activity: Activity;
};

export type Activity =
  | VideoActivity
  | BlankActivity
  | ColoringActivity
  | DiaryActivity;

export type VideoActivity = {
  type: 'video';
  videoUrl: string;
  instruction?: string;
};

export type BlankActivity = {
  type: 'blank';
  prompt: string;
  image?: string;
  questions: Array<{ template: string; answers: string[] }>;
};

export type ColoringActivity = {
  type: 'coloring';
  templateImage: string;
  exampleImage?: string;
  palette?: string[];
};

export type DiarySection =
  | { kind: 'radio'; key: string; label: string; options: string[] }
  | { kind: 'checkbox'; key: string; label: string; options: string[] }
  | { kind: 'text'; key: string; label: string; placeholder?: string }
  | { kind: 'textarea'; key: string; label: string; placeholder?: string; rows?: number };

export type DiaryActivity = {
  type: 'diary';
  sections: DiarySection[];
};
```

### 7.3 콘텐츠 로더

`src/lib/content.ts`:

```ts
import type { Issue, Page } from './types';
import issueData from '../data/issues/5-4.json';

export async function fetchIssue(): Promise<Issue> {
  // 향후 API 전환 시 이 함수 내부만 교체
  return issueData as Issue;
}

export async function fetchPage(day: number, pageNo: string): Promise<Page | null> {
  const issue = await fetchIssue();
  const targetDay = issue.days.find((d) => d.day === day);
  return targetDay?.pages.find((p) => p.pageNo === pageNo) ?? null;
}
```

### 7.4 저장소 추상화

`src/lib/storage.ts`:

```ts
import { get, set } from 'idb-keyval';

const ISSUE_ID = '5-4';

export const progressStore = {
  async getDone(pageNo: string): Promise<boolean> {
    return localStorage.getItem(`progress:${ISSUE_ID}:${pageNo}`) === 'true';
  },
  async setDone(pageNo: string, done: boolean): Promise<void> {
    localStorage.setItem(`progress:${ISSUE_ID}:${pageNo}`, String(done));
  },
};

export const inputStore = {
  async get<T>(pageNo: string): Promise<T | null> {
    const raw = localStorage.getItem(`input:${ISSUE_ID}:${pageNo}`);
    return raw ? (JSON.parse(raw) as T) : null;
  },
  async set<T>(pageNo: string, value: T): Promise<void> {
    localStorage.setItem(`input:${ISSUE_ID}:${pageNo}`, JSON.stringify(value));
  },
};

export const coloringStore = {
  async get(pageNo: string): Promise<Blob | undefined> {
    return get<Blob>(`coloring:${ISSUE_ID}:${pageNo}`);
  },
  async set(pageNo: string, blob: Blob): Promise<void> {
    await set(`coloring:${ISSUE_ID}:${pageNo}`, blob);
  },
};
```

## 부록: 트러블슈팅

### `npm create vite@latest .` 실행 시 폴더가 비어있지 않다는 경고
→ `_readme/` 때문입니다. `Ignore files and continue` 선택하면 보존됩니다.

### Tailwind 클래스가 적용되지 않음
→ `tailwind.config.js`의 `content` 배열에 `'./src/**/*.{ts,tsx}'`가 있는지, `src/index.css`에 `@tailwind` 지시문 3개가 있는지, `main.tsx`에서 `index.css`를 import 하고 있는지 확인.

### `code` 명령이 동작하지 않음
→ VSCode를 열고 `Ctrl+Shift+P` → "Shell Command: Install 'code' command in PATH" 실행 후 PowerShell 재시작.

### 한글 파일명이 깨짐
→ PowerShell 인코딩 확인: `$OutputEncoding = [System.Text.Encoding]::UTF8`. `.editorconfig`의 `charset = utf-8`도 영향.

### `react-konva`가 빌드 에러
→ `konva`를 별도로 설치했는지 확인 (`npm install konva`). PoC 단계에 색칠이 단순하면 native Canvas로 대체 가능.
