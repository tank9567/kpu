# 21. 영상 활동 컴포넌트 (VideoActivity)

> 4종 활동 중 첫 번째 — 가이드 영상을 시청하며 따라하는 활동.

## 개요

| 항목 | 값 |
|------|-----|
| 활동 타입 | `video` |
| 대표 페이지 | `1_1.1 국민 건강 체조` |
| 메인 인터랙션 | YouTube 영상 시청 + 완료 체크 |
| 사용 라이브러리 | `react-youtube` |

---

## 활동 데이터 스키마

```ts
type VideoActivity = {
  type: 'video';
  videoUrl: string;        // YouTube URL (youtu.be / watch?v= / embed 모두 지원)
  instruction?: string;    // 활동 안내 문구
};
```

샘플 (`app/src/data/issues/5-4.json` Day 1):
```json
{
  "type": "video",
  "videoUrl": "https://youtu.be/K6tHparZ5ek",
  "instruction": "활기찬 국민 건강체조 노래를 들으며, 긴장된 근육을 이완하고 신나게 체조를 즐겨보세요."
}
```

---

## 사용자 흐름

1. 홈에서 Day 1 → "국민 건강 체조" 클릭
2. 페이지 상단에 안내 문구(노란색 박스)
3. YouTube 플레이어가 16:9 비율로 임베드되어 자동 표시
4. 영상 시청 (사용자가 직접 재생 버튼)
5. 시청 완료 후 하단의 **"오늘의 활동 완료"** 버튼 클릭
6. 완료 상태가 `localStorage`에 저장되고 0.8초 후 홈으로 자동 이동

---

## 컴포넌트 인터페이스

```tsx
<VideoActivity

  activity={{

/>
```

## 핵심 구현 사항

### YouTube URL 파싱
다양한 형식의 URL에서 video ID를 추출하는 `extractYouTubeId()` 헬퍼:
- `https://youtu.be/XXX` → `XXX`
- `https://www.youtube.com/watch?v=XXX&list=...` → `XXX`
- `https://www.youtube.com/embed/XXX` → `XXX`

### Player 옵션
- `rel: 0` — 관련 영상 노출 최소화
- `modestbranding: 1` — YouTube 로고 최소화
- 컨테이너에 `aspect-video` Tailwind 유틸리티 사용 → 16:9 비율 고정

### 완료 처리
완료 버튼 누르면 `progressStore.setDone(pageNo, true)` 호출 → `localStorage`에 `progress:5-4:1_1.1 = true` 저장.

## 검증 방법

- [ ] 홈 → Day 1 → "국민 건강 체조" 진입 시 안내 문구가 노란 박스로 보임
- [ ] YouTube 플레이어가 정상 로드 (재생 가능)
- [ ] 완료 버튼 클릭 시 "✓ 완료했어요" 변경 + 0.8초 후 홈 이동
- [ ] 홈에 돌아오면 해당 항목의 체크 아이콘이 ✓로 표시
- [ ] 페이지 재진입 시에도 완료 상태 유지

## 가이드 영상과의 관계

영상 활동(`type: 'video'`) 페이지에서는 **활동의 본체 자체가 영상**이므로 별도의 `Page.guideVideoUrl`을 사용하지 않습니다. `activity.videoUrl`이 곧 사용자가 시청할 영상입니다.

빈칸·색칠·일기 활동 페이지에서는 `Page.guideVideoUrl`(활동 안내 영상)이 별도로 표시됩니다. → [22](./22.%20빈칸%20활동%20컴포넌트.md) / [23](./23.%20색칠%20활동%20컴포넌트.md) / [24](./24.%20일기%20활동%20컴포넌트.md) 참조.

## 알려진 제약 / 후속 작업

1. **영상 시청 시간 추적 안 함** — 완료 버튼을 누르면 시청 여부와 무관하게 완료 처리. 향후 `onEnd` 이벤트로 자동 완료 옵션 추가 가능.
2. **자막 자동 표시 X** — YouTube 자막 옵션을 자동으로 켜지 않음 (`cc_load_policy` 미설정). 어르신 사용성 테스트 후 결정.
3. **속도 조절 안내 없음** — 0.75x 등 느린 속도가 도움될 수 있으나 별도 UI 없음.
4. **URL 미확정 영상** — 5호 4주차 콘텐츠의 1_2.1 이후 URL이 미확정 ([13. 콘텐츠 자산 목록](./13.%20콘텐츠%20자산%20목록.md) 참조). 현재 PoC에선 1_1.1만 실제 URL 적용. 가이드 영상 URL들도 임시 더미값으로, 영상이 실제 업로드되면 자동 반영됨.
